Class AuthRequest

source code

Base class to parse, validate, and generate auth requests going to the server. Mostly it will be used in the generate mode. The aim is to simplify writing applications around the auth request. We could potentially use this with AppEngine and Django that are python-based. This interface supports only v1.5 and public AuAs

Set the configuration, flag to indicate whether this is biometrics or demographics, and additional addtributes
