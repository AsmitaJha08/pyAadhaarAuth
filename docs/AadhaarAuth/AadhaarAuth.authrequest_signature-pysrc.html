 31  # Signs a file using a dynamicaly created template, key from PEM file and 
 32  # an X509 certificate. The signature has one reference with one enveloped  
 33  # transform to sign the whole document except the <dsig:Signature/> node  
 34  # itself. The key certificate is written in the <dsig:X509Data/> node. 
 35  # 
 36  # This example was developed and tested with OpenSSL crypto library. The  
 37  # certificates management policies for another crypto library may break it.

 39  # Usage:

 41  #

 43  #       ./sign3.py sign3-doc.xml public.p12 hello > sign3-res.xml 
 44  # 
 45  # The result signature could be validated using verify3 example:
